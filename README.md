# QtileDesktop Instalacion

Lo primero que tenemos que instalar en nuestro sistema linux es Qtile lo haremos de la siguiente manera.

    - Habriremos una terminal y pondremos lo siguiente:
      - `sudo pacman -S qtile`
    - Con eso instalaremos Qtile por defecto veremos que no tenemos nada hasi que nos descargaremos este repositorio
    y lo pondremos en la ruta **/home/USUARIO/.config/qtile** 

    - Una vez realizado esto ya tendremos nuestro tema instalado para nuestro escritorio.
